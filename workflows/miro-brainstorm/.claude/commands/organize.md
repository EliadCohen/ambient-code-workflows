# /organize - Organize Ideas by Theme

## Purpose

Analyze all tracked ideas and organize them into semantic themes. Create visual organization on the Miro board with theme markers and dependency diagrams.

## Prerequisites

- Ideas have been tracked in the table
- At least 5+ ideas to cluster

## Process

1. Read all ideas from tracking table
2. Perform semantic analysis to identify themes
3. Create theme markers/documents on Miro board
4. Create a flowchart diagram showing:
   - Theme relationships
   - Dependencies between themes
   - Implementation phases
5. Generate theme breakdown document

## Semantic Clustering

Group ideas by:
- **Purpose**: Distribution, security, optimization, etc.
- **Technology**: AI/ML, infrastructure, tooling
- **Scope**: Foundation, advanced features, alternatives
- **Difficulty**: Low/Medium/Insane groupings

## Output

- Theme marker documents on Miro board
- Flowchart diagram showing relationships
- `artifacts/miro-brainstorm/THEMES.md` - Detailed theme analysis
